# PayLeaks Landing Page

## Run Instructions

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Configuration

### Countdown Timer
- Edit `COUNTDOWN_START_SECONDS` in `app/page.tsx` (line 6)
- Default: 300 seconds (5 minutes)

### Reviews
- Edit the `reviews` array in `app/page.tsx` (lines 8-15)
- Keep reviews short (1-2 sentences max)
